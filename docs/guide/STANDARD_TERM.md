# 표준 용어 사전

- 프로젝트 관련 표준 용어만을 기재할 것
- 단어/약어/영문명이 없으면 공백으로 기재

# (1). 회원/계정 관련

| 용어            | 영문약어 | 영문명                  | 정의                   | 기타             |
| --------------- | -------- | ----------------------- | ---------------------- | ---------------- |
| 회원가입        | MB-SU    | Member Sign Up          | 회원이 되는 절차       | 이메일/소셜 포함 |
| 로그인          | MB-LI    | Member Login            | 회원 계정 접속         | -                |
| 로그아웃        | MB-LO    | Member Logout           | 계정 연결 종료         | -                |
| 비밀번호 재설정 | MB-RPW   | Member Password Reset   | 비밀번호 초기화 절차   | 인증 필요        |
| 프로필 설정     | MB-PFS   | Member Profile Settings | 개인 정보 및 목표 관리 | 사진/닉네임 포함 |
| 계정 연동       | MB-ACC   | Member Account Linking  | 소셜/외부 서비스 연결  | 구글/애플 로그인 |
