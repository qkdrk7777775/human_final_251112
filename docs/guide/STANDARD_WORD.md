# 표준 단어 사전

- 서비스 관련 표준 단어만을 기재할 것
- 단어/약어/영문명이 없으면 공백으로 기재

## (1). 서비스 기능 관련

| 단어     | 영문약어 | 영문명       | 정의             | 기타    |
| -------- | -------- | ------------ | ---------------- | ------- |
| 홈       | HM       | Home         | 첫 화면          | -       |
| 회원     | MB       | Member       | 서비스 이용자    | -       |
| 계정     | AC       | Account      | 로그인 정보 단위 | -       |
| 프로필   | PF       | Profile      | 개인 정보 화면   | -       |
| 구독     | SUB      | Subscription | 정기 결제 상태   | 월/연간 |
| 결제     | PY       | Payment      | 비용 지불 행위   | -       |
| 리포트   | RP       | Report       | 분석 결과 화면   | -       |
| 알림     | NT       | Notification | 메시지 안내      | -       |
| 검색     | SR       | Search       | 콘텐츠 찾기 기능 | -       |
| 필터     | FT       | Filter       | 조건별 분류 기능 | -       |
| 설정     | ST       | Settings     | 환경 조정 메뉴   | -       |
| 다운로드 | DL       | Download     | 저장 기능        | -       |
| 스트리밍 | STG      | Streaming    | 실시간 재생      | -       |
